# 种子用户访谈 + 试用观察提纲

> 目的：用最少的人、最真实的反馈，回答一个问题——**这东西父母/果农到底用不用、有没有用。**
> 用法：找 5-10 个真实果农（父母 + 他们的果农熟人）。**先让他们用，再问；多看少说，别替他们答。**

---

## 北极星（只看这一个）

> **他们会不会"主动"再打开第二次？**

其余都是辅助信号：拍照诊断用了几次、预警有没有救过一次损失、有没有人主动说"这个有用"。
**如果三个月后只有你在用、果农从不主动碰 → 说明定位错了，要回头想。**

---

## 一、用之前（5 分钟，了解背景）

1. 您种了多少年果树？种的啥、多大面积？
2. 平时遇到不懂的（比如不知道是什么病、该不该打药），您一般问谁、怎么解决？
3. 用手机多吗？平时用微信干啥？会自己装 App 吗，还是孩子帮装？
4. 最头疼、最怕的是哪种情况？（霜冻？病虫？卖不上价？）

## 二、让他用（**关键：你闭嘴，只观察**）

把手机给他，说一句"您随便点点看"，然后**记下**：
- [ ] 他第一眼看哪里？看得懂首页吗？
- [ ] 字够不够大、按钮好不好点？（戴老花镜/太阳下）
- [ ] 让他**拍一张自家的叶子/果子诊断**——能不能独立完成？卡在哪一步？
- [ ] 结果他看懂了吗？"用药要问农技员"会不会让他困惑或失望？
- [ ] 点"听"——声音、语速、口音能听懂吗？
- [ ] 让他**语音问一个问题**（陕西话）——识别准不准？识别错了他会不会改用打字？
- [ ] 全程**他需要你在旁边提示几次**？哪几步必须有人教？

> 记录原则：**写他的原话和动作，不写你的解释。** "他盯着屏幕 10 秒没动，问'这咋弄'" 比 "操作有点难" 有用 100 倍。

## 三、用之后（5 分钟，问感受）

1. 刚才哪个功能您觉得**有用**？哪个**没用/看不懂**？
2. 如果这是您自己的手机，您**愿意自己再打开用一次**吗？什么情况下会用？
3. 您会**推荐给一起种地的人**吗？为什么？
4. 如果要花钱（比如一年几十块），您**愿意**吗？多少能接受？（看反应，不强求）
5. 您最希望它能帮您做的、但现在没有的，是什么？

## 记录模板（每人一份）

```
受访人：__（年龄/种啥/面积）__   日期：__
背景：平时遇到病虫怎么解决：__
观察卡点（按发生顺序）：
  1. 在__步，现象__，原话"__"
  2. ...
最有用的功能：__    最没用/看不懂：__
愿不愿主动再用：__   愿不愿推荐：__   付费意愿：__
他想要但没有的：__
我的判断：__（这人是真实用户吗？这功能成立吗？）
```

---

## 访谈纪律（容易犯的错）

- ❌ 别问"你觉得这个 App 好不好"——人都会客气说好。**问行为**（用没用、会不会再用、愿不愿推荐）。
- ❌ 别引导/替他操作——你一插手，就看不到真实门槛。
- ❌ 别解释功能多牛——你不在场时没人解释，看他能不能自己懂。
- ✅ 多问"为什么"和"然后呢"。
- ✅ 5-10 个就够看出规律，不用追求数量。
