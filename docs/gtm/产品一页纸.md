# AI 助农 · 一页纸介绍

> 用途：给果农 / 合作社 / 农技站 / 农技员 / 潜在合伙人快速讲清"这是个啥、帮谁、帮啥"。
> 可裁剪成微信推文、海报文案、合作洽谈开场。

---

## 一句话

**给陕西果农的"会看病、会提醒、会算账"的手机助手——拍张照就知道是什么病虫，霜冻冰雹来之前主动喊你，遇事还能一键打给农技员。**

## 给谁用

- **核心**：陕西种苹果、桃（以及梨、樱桃）的果农，尤其**年纪大、不爱打字**的。
- **帮手**：他们的**子女**——在外地也能远程帮父母建园子、看预警、配置。

## 解决什么痛点

| 果农的难处 | 我们怎么帮 |
|---|---|
| 不知道是啥病、要不要打药 | **拍照诊断**：拍叶子/果子 → 说出最可能的病虫 + 农业防治；用药导向农技员，**不乱开药** |
| 霜冻、冰雹猝不及防，一夜绝收 | **灾害预警**：结合本地天气和果树生长期，**来之前主动推送**到手机，并给应对办法 |
| 啥时候该疏果、套袋、施肥记不清 | **农事日历**：按本地物候期，告诉你"这阵子该干啥" |
| 不会打字、看不清小字 | **语音问答 + 大字大按钮 + 朗读**，问一句就答 |
| 一年到头不知道赚没赚 | **园子账本**：记投入和收入，年底算清净收益 |

## 凭什么比通用 App 强（差异化）

1. **超本地**：物候期、病虫害、用药都按**陕西关中/渭北**实际来，不是全国一个样。
2. **适老化**：为不会打字的老人设计——语音、大字、朗读、一键拨号。
3. **子女代管**：一个产品两种用法，子女远程帮父母兜底。
4. **用药有底线**：AI 不乱推农药，化学用药一律导向农技员，**安全第一**。
5. **越用越准**：可灌入权威农技资料、可由农技员审核，回答有据可依。
